CURSOS DE BANCO DE DADOS

ORACLE

Neste curso de Oracle, você aprenderá sobre o banco de dados Oracle Database e como usá-lo para armazenar, recuperar e manipular dados com SQL. Ao final, estará preparado para utilizar o Oracle Database em seus projetos.

SQL SERVER

Neste curso de SQL Server, você aprenderá sobre o sistema de gerenciamento de banco de dados relacional da Microsoft. Vamos explorar desde conceitos básicos até tópicos avançados, incluindo a criação de bancos de dados, consultas SQL, procedimentos armazenados e muito mais. Ao final, você estará preparado para usar o SQL Server para armazenar e gerenciar dados de forma eficiente.

  • Assistir

MYSQL

Neste curso de MySQL, você aprenderá sobre um dos sistemas de gerenciamento de banco de dados mais populares do mundo. Vamos abordar desde os conceitos básicos até tópicos avançados, incluindo a criação de bancos de dados, consultas SQL, otimização de consultas e segurança do banco de dados. Ao final do curso, você estará apto a utilizar o MySQL para gerenciar eficientemente seus dados.

MONGODB

Neste curso de MongoDB, você vai explorar um dos bancos de dados mais populares entre desenvolvedores de aplicativos modernos. Vamos abordar desde os conceitos básicos até tópicos avançados, incluindo a modelagem de dados, consultas, indexação e escalabilidade. Ao final do curso, você estará preparado para utilizar o MongoDB de forma eficiente em seus projetos.

  • Assistir

POSTGRESQL

Neste curso de PostgreSQL, você vai aprender sobre um poderoso sistema de gerenciamento de banco de dados relacional de código aberto. Vamos abordar desde os conceitos básicos até tópicos avançados, incluindo a instalação e configuração, modelagem de dados, consultas SQL complexas, otimização de desempenho e segurança. Ao final do curso, você estará apto a utilizar o PostgreSQL de forma eficaz em seus projetos de banco de dados.
